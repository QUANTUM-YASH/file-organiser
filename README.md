# File Organizer

A **Python script** to automatically organize files in any folder into categories like Images, PDFs, Audio, Videos, and Others.  
Includes **undo functionality** and a **cross-platform folder picker GUI**.

---

## Features

- Organizes files by type:
  - Images: `.jpg, .jpeg, .png`
  - PDFs: `.pdf`
  - Audio: `.mp3, .wav`
  - Videos: `.mp4, .mkv`
  - Others: Anything else
- Undo last organize action
- Works on **Windows, Mac, Linux**
- Easy folder selection via GUI
